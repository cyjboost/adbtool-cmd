### *adbtool-cmd*
只是一个cmd的脚本  
English users,[click here](https://github.com/cyjboost/adbtool-cmd/English)     
### 使用      
首先找个位置存放脚本，在同级目录下创建bin文件夹    
在bin文件夹中放入所需文件即可(adb,fastboot,scrcpy以及其所需的所有dll)
